GAME BOY Yourself (Beta)

Upload an image and watch it on a classic Nintendo Game Boy screen style. It is a tiny experimental converter that transforms your image with the aspect of this classic 8-bit video game console.

Twittear
  1. Choose an Image from your computer
    It should be less than 600KB
  2. Wait until the image is displayed (be patient) and Download!

How it works?

This experiment uses no server-side processing. When you upload an image your javascript browser engine determines the color of each pixel and selects the closest green tone from a default Game Boy color palette. Time to acomplish the task depends on the image size and resolution (it's directly related to the amount of pixels to convert). Once finish it is shown inside an HTML5 canvas.
